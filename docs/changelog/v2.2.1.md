# v2.2.1 - 2026-03-16

> Web Dashboard, OpenRouter Provider, Ollama Improvements, Code Splitting

---

## Highlights

- **Web Dashboard** — Full-featured React 19 + Vite + Tailwind SPA at `localhost:18920/web/`
- **OpenRouter Provider** — Free models (Hunter Alpha, Nemotron 3 Super) via native Anthropic-compatible API
- **Ollama Native Thinking** — Extended thinking protocol support with proper thinking block handling
- **Auto-Update Check** — Non-blocking CLI update notifications with 24-hour cache
- **Code Splitting** — Modularized proxy control modules for better maintainability

---

## Features

### Web Dashboard

Standalone dashboard at `http://localhost:18920/web/` — auto-starts with `omc cc`, or start manually:

```bash
omc proxy dashboard       # Start dashboard on port 18920
omc proxy dashboard --stop  # Stop dashboard
```

- **Session Browser** — Browse Claude Code conversations, rename (manual + AI-powered), delete, bulk cleanup empty stubs
- **Memory Browser** — OMC project memories + CC auto-memory + global memories, with inline editing
- **Preferences Manager** — View, add (global/project-scoped), delete preference rules
- **AI-Powered Tools** — Bulk AI session rename, memory operations (compact/daily/summarize/clear) with two-phase Analyze → Execute flow
- **Per-Session Model Switch** — Select any active proxy session, switch its model independently
- **Models Registry Editor** — Full CRUD for models — add, edit, delete models per provider
- **Providers Page** — View all configured providers with status, API key env var, base URL
- **Light/Dark Theme** — Toggle between dark, light, and auto (system preference) modes
- **Proxy Instance Registry** — Per-session proxy instances with heartbeat, health, uptime tracking

### OpenRouter Provider

- **Native Anthropic Messages API** — Uses `/api/messages` endpoint with Bearer auth for full protocol support
- **Free Models** — `openrouter/hunter-alpha` (1T params, 1M context) and `nvidia/nemotron-3-super-120b-a12b:free` (120B MoE, 1M context)
- **Sanitizer Module** — Thinking block stripping, tool cap (64), unsupported content type filtering
- **Metadata Truncation** — `metadata.user_id` truncated to 128 chars to comply with OpenRouter limits

### Ollama Improvements

- **Native Thinking Protocol** — Proxy keeps `anthropic-beta` header for Ollama, skips thinking sanitization
- **Thinking Model Fallback** — Falls back to `thinking` content blocks when no `text` blocks returned
- **Dummy API Key** — Auto-generated for local Ollama instances to prevent auth errors
- **Thinking Content in Dashboard** — Reasoning traces forwarded via `thinking` response field

### Auto-Update Check

- Two-phase design: sync banner from cache (zero latency) + detached background fetch for next run
- Stable: checks npm registry; Beta: checks GitHub API for dev branch commits
- Cache at `~/.claude/oh-my-claude/.update-check.json` with 24-hour interval

## Bug Fixes

- **MCP Discovery on Linux** — Mirror MCP config to `~/.claude.json` for dual-config environments
- **tmux Detection on Linux** — Uses `tmux -V` (installed check) instead of `tmux list-sessions` (requires running server)
- **AI Session Rename** — Increased `max_tokens` from 100 to 1024 for thinking model compatibility
- **Dashboard Favicon** — Added inline SVG favicon, eliminating 404 on every page load
- **Menubar Build** — Git LFS pointer detection, Cargo toolchain validation, improved Bun path resolution
- **Beta Update Build** — Run `bun install` before building to install workspace devDependencies
- **Daily Memory Operation** — Properly consolidates same-date sessions with full content, deletes originals
- **YAML Frontmatter Parsing** — Fixed quote stripping in frontmatter values
- **Statusline 1M Context** — Fixed context window detection for Opus 4.6 with `[1m]` suffix
- **OpenRouter Prefix Dedup** — Fixed double-prefixing in proxy logs and menubar tray
- **WezTerm Git Bash** — Fixed `find_git_bash()` on Windows with non-standard Git paths
- **Dashboard Project Discovery** — JSONL `cwd` extraction scanned only first 10 lines, missing projects where `file-history-snapshot` entries dominated early lines. Increased to 200 lines for both memory and preferences scanners
