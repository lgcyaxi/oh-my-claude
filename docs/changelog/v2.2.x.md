# v2.2.x Changelog

## v2.2.9 (2026-04-09)
### Changes
- `/omcx-commit` now commits directly without asking for confirmation (invoking the command is the confirmation)
- Installer refactored from 1428-line monolith into 9 focused sub-modules under `src/cli/installer/`

- Removed `-a`/`-auto` shortcuts from CLI
- Fixed statusline stale model display after switching
- Fixed terminal detection spawn error handling
