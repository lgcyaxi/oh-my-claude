# v1.0.0

> Released: 2025-01-15

First public release of oh-my-claude - Multi-provider MCP server for Claude Code with specialized agent workflows.

## Highlights

- **Multi-Provider Architecture**: Route tasks to DeepSeek, ZhiPu GLM, MiniMax via Anthropic-compatible APIs
- **Specialized Agents**: 9 pre-configured agents for different task types
- **Official MCP Setup**: One-command installation for Sequential Thinking, MiniMax, and GLM MCPs
- **Planning System**: Strategic planning with Prometheus agent

## Features

### Core

- Multi-provider MCP server with Anthropic-compatible API support
- Background task execution with concurrent task management
- Per-provider rate limiting and parallel execution
- Flexible configuration via `~/.claude/oh-my-claude.json`

### Agents

| Agent | Provider | Role |
|-------|----------|------|
| Sisyphus | Claude | Primary orchestrator |
| Claude-Reviewer | Claude | Code review, QA |
| Claude-Scout | Claude | Fast exploration |
| Prometheus | Claude | Strategic planning |
| Oracle | DeepSeek | Deep reasoning |
| Librarian | ZhiPu | External research |
| Explore | DeepSeek | Codebase search |
| Frontend-UI-UX | ZhiPu | Visual/UI design |
| Document-Writer | MiniMax | Documentation |

### Slash Commands

**Agent Commands (`/omc-*`)**:
- `/omc-sisyphus` - Full implementation orchestrator
- `/omc-oracle` - Deep reasoning and architecture
- `/omc-librarian` - External research and docs
- `/omc-reviewer` - Code review and QA
- `/omc-scout` - Fast exploration
- `/omc-explore` - Codebase search
- `/omc-plan` - Strategic planning with Prometheus
- `/omc-start-work` - Begin work on existing plan

**Quick Action Commands (`/omcx-*`)**:
- `/omcx-commit` - Smart git commit with conventional format
- `/omcx-implement` - Implement feature with best practices
- `/omcx-refactor` - Refactor code with quality improvements
- `/omcx-docs` - Generate or update documentation

### CLI

```bash
# Installation
oh-my-claude install
oh-my-claude install --force
oh-my-claude install --skip-mcp

# Status & Diagnostics
oh-my-claude status
oh-my-claude doctor
oh-my-claude doctor --detail

# MCP Server Setup
oh-my-claude setup-mcp
oh-my-claude setup-mcp --list
oh-my-claude setup-mcp --thinking
oh-my-claude setup-mcp --minimax
oh-my-claude setup-mcp --glm

# Uninstall
oh-my-claude uninstall
```

### Official MCP Servers

The `setup-mcp` command installs:
- **sequential-thinking** (Anthropic) - Structured problem-solving
- **MiniMax** - Coding plan assistance
- **web-reader** (ZhiPu) - Web content extraction
- **web-search-prime** (ZhiPu) - Web search
- **zread** (ZhiPu) - GitHub repository reader
- **zai-mcp-server** (ZhiPu) - Image/video analysis
