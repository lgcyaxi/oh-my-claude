# v2.2.4 Changelog

## v2.2.4 - 2026-04-01

### Fixes

- **Dashboard OMC Mem Duplicate Entries** — Fixed `findOmcProjectMemDirs()` reading the same project memory directory multiple times when multiple Claude project folders resolve to the same cwd. Added three-layer deduplication: backend cwd dedup (root cause), API response ID dedup (defense in depth), and frontend ID dedup (safety net). Previously caused all projects to show the same accumulated list and unreliable click behavior due to duplicate React keys corrupting DOM reconciliation.
