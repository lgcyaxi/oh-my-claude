# v2.2.6 Changelog

## v2.2.6 - 2026-04-02

### Improvements

- **MiniMax M2.7 Default** — Updated default MiniMax model from M2.5 to M2.7 across all providers, agents, and aliases. `omc -p mm-cn` now uses the latest M2.7 model. M2.5 remains available as a selectable option in the model registry.

### Fixes

- **Thinking Signature Error Recovery** — Added automatic retry in the proxy passthrough handler when Anthropic returns 400 "Invalid `signature` in `thinking` block". The proxy now strips thinking blocks and retries the request transparently, preventing intermittent failures caused by stale or corrupted thinking block signatures.
