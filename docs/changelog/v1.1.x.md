# v1.1.x Changelog

## v1.1.4 - 2026-01-16

Major feature release with companion tools, config management, and enhanced statusline.

### Added

- **`setup-tools` command**: Interactive menu to install companion tools (CCometixLine)
- **`config` command**: CLI to manage oh-my-claude configuration (get/set/unset/list)
- **`cleanup` command**: Clean up stale session data and temporary files
- **`execute_agent` MCP tool**: Blocking agent execution without polling clutter
- **Beta channel support**: `update --beta` to install from GitHub dev branch
- **Enhanced StatusLine**: ANSI colors, animated spinners, provider/model display

### Fixed

- Per-session status tracking (sessions no longer shared between terminals)
- CCometixLine integration (`ccline --init` instead of `ccline install`)
- Task agent tracking in statusline
- `/omc-plan` command now properly invokes Prometheus agent
