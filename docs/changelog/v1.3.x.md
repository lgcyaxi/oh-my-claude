# v1.3.x Changelog

## v1.3.0-beta.0 - 2026-01-29

### Added

- **Output Style Manager**: CLI commands to manage Claude Code output styles
  - `oh-my-claude style list` - List available built-in and custom styles
  - `oh-my-claude style set <name>` - Switch active output style
  - `oh-my-claude style show [name]` - Display style content
  - `oh-my-claude style reset` - Reset to Claude Code default
  - `oh-my-claude style create <name>` - Create custom style from template
- **Built-in Style Presets** (5 styles):
  - `engineer-professional` - SOLID/KISS/DRY/YAGNI principles
  - `agent` - Autonomous agent mode, minimal narration
  - `concise-coder` - Code-first, no explanations
  - `teaching` - Educational with concepts and trade-offs
  - `review` - Code review with severity levels
- **Style Deployment**: Installer now deploys built-in styles to `~/.claude/output-styles/`
- **StatusLine**: New abbreviations for built-in styles (agent, coder, teach, review)

### Changed

- Installer reports deployed output styles during installation
- Documentation convention added to CLAUDE.md for feature updates
- `setup-tools` command now includes UI UX Pro Max skill installation
- `doctor --detail` shows companion tools status (UI UX Pro Max, CCometixLine, Output Styles)
