# v2.2.1-beta.1 - 2026-03-12

> OpenRouter provider support with free models

---

## Highlights

- **OpenRouter Provider** — Added OpenRouter as a new provider with two free models:
  - `openrouter/hunter-alpha` — Hunter Alpha (free, 1T params, 1M context, agentic)
  - `nvidia/nemotron-3-super-120b-a12b:free` — Nemotron 3 Super (free, 120B hybrid MoE, 1M context)

---

## Features

- **OpenRouter Integration** — OpenAI-compatible API with base_url `https://openrouter.ai/api/v1`
- **Provider Aliases** — `or` and `openrouter` as shortcuts for provider switching
- **Environment Variable** — `OPENROUTER_API_KEY` for authentication

---

## Migration Notes

### From v2.2.0

No breaking changes. Simply add `OPENROUTER_API_KEY` to use OpenRouter:

```bash
export OPENROUTER_API_KEY=your-openrouter-api-key
```
