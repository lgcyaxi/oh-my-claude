# v1.2.2-beta.1

Release date: 2026-01-17

## Changes

### Removed Unused Concurrency System

Removed the dead code concurrency control system that was never actually used:
- Deleted `src/mcp/background-agent-server/concurrency.ts`
- Removed `concurrency` config from schema (no longer needed in `oh-my-claude.json`)
- Removed concurrency display from statusline (`DS: 10 ZP: 10 MM: 5` no longer shown)
- Removed "Concurrent Execution" from feature list in README

The `withConcurrencyLimit` function was exported but never called anywhere. External API rate limits are handled server-side by the providers themselves.

### Enhanced StatusLine Display

Improved statusline to show richer information during MCP agent execution:

**New format:**
```
omc [⠙ Oracle: 32s DS/R] "Analyze the code..."
```

**Components:**
- Animated spinner (⠋⠙⠹⠸⠼⠴⠦⠧⠇⠏) - rotates every 80ms
- Agent name with color coding (cyan for MCP, yellow for Task tool)
- Elapsed time
- Provider/Model abbreviations (e.g., `DS/R` = DeepSeek/Reasoner)
- Truncated prompt preview (max 30 chars)

**Model abbreviations:**
- DeepSeek: `R` (reasoner), `C` (chat)
- ZhiPu: `G4` (glm-4.7), `V` (glm-4v-flash)
- MiniMax: `M2` (MiniMax-M2.1)
