# v1.2.2-beta.5 - 2026-01-23

## New Feature: Segment-Based Statusline

A comprehensive statusline enhancement inspired by CCometixLine and ccstatusline, with configurable segments and semantic colors.

### Segments Available

| Segment | Description | Data Source |
|---------|-------------|-------------|
| **Model** | Current Claude model | Claude Code stdin (graceful degradation) |
| **Git** | Branch name + dirty/clean status | Shell commands |
| **Directory** | Project name (folder basename) | Environment |
| **Context** | Token usage percentage | Transcript parsing (graceful degradation) |
| **Session** | Session duration | Session directory |
| **Output Style** | Current output mode | Claude Code stdin (graceful degradation) |
| **MCP** | Background task status | Existing MCP tracking |

### Configuration

New config file at `~/.config/oh-my-claude/statusline.json`:

```json
{
  "enabled": true,
  "preset": "standard",
  "segments": {
    "model": { "enabled": false, "position": 1 },
    "git": { "enabled": true, "position": 2 },
    "directory": { "enabled": true, "position": 3 },
    "context": { "enabled": false, "position": 4 },
    "session": { "enabled": true, "position": 5 },
    "output-style": { "enabled": false, "position": 6 },
    "mcp": { "enabled": true, "position": 7 }
  },
  "style": {
    "separator": " ",
    "brackets": true,
    "colors": true
  }
}
```

### Presets

- **minimal**: Git, Directory only
- **standard**: Git, Directory, Session, MCP (default)
- **full**: All segments enabled

### Semantic Colors

- 🟢 **Green** (good): Clean git status, low token usage
- 🟡 **Yellow** (warning): Dirty git, long session (>1h), moderate token usage
- 🔴 **Red** (critical): Many changes, very high token usage
- 🔵 **Cyan** (neutral): Directory, general info

### Example Output

```
omc [dev*] [oh-my-claude] [45m] [⠙ Oracle: 32s DS/R]
     │          │          │            │
     │          │          │            └─ MCP background task
     │          │          └─ Session duration
     │          └─ Project name
     └─ Git branch (dirty)
```

```
    directory.ts                # Directory segment (project name only)

    model.ts                    # Model segment
    context.ts                  # Context/tokens segment
    output-style.ts             # Output style segment
```

## Technical Details

- **Graceful Degradation**: Advanced segments (Model, Context, Output Style) show `[?]` when Claude Code data is unavailable
- **100ms Timeout**: Statusline script has hard timeout to prevent blocking terminal
- **Backward Compatible**: Existing MCP task display preserved in MCP segment
- **Auto-Install**: Default config created during `oh-my-claude install`
