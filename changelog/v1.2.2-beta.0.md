# v1.2.2-beta.0

Release date: 2026-01-17

## New Features

### Rich Context System for MCP Background Agents

Added automatic context gathering and injection for MCP background agents. Agents now receive relevant project context to improve response quality.

#### Context Types
- **project-structure**: Directory tree (depth 3, excludes node_modules, .git, etc.)
- **package-info**: package.json dependencies and scripts
- **git-status**: Current branch and uncommitted changes
- **config-files**: Detection of tsconfig.json, eslint, vite, etc.
- **readme**: Project README.md content
- **recent-changes**: Last 10 git commits
- **test-patterns**: Test directory detection

#### Smart Detection
Context is automatically selected based on keywords in your prompt:
- "architecture", "structure" → project-structure, config-files
- "bug", "fix", "error" → related-files, git-status
- "test", "spec" → test-patterns, related-files
- "dependency", "package" → package-info
- "commit", "branch" → git-status, recent-changes

#### Per-Agent Profiles
Each agent has its own context preferences and token budget:
- **Oracle**: 4000 tokens, prioritizes project-structure and config
- **Librarian**: 1500 tokens, prioritizes package-info
- **Document-Writer**: 3000 tokens, prioritizes readme and structure

#### New Tool Parameters
Both `execute_agent` and `launch_background_task` now accept optional `context_hints`:
```json
{
  "context_hints": {
    "keywords": ["additional", "keywords"],
    "file_patterns": ["src/**/*.ts"],
    "skip_context": false
  }
}
```

## New Files

- `src/context/types.ts` - Core type definitions
- `src/context/detector.ts` - Keyword-based context detection
- `src/context/gatherer.ts` - Context collection functions
- `src/context/formatter.ts` - Context formatting for prompts
- `src/context/index.ts` - Module exports
- `src/agents/context-profiles.ts` - Per-agent context preferences

## Technical Changes

- Updated `task-manager.ts` to enrich prompts with gathered context
- Updated `server.ts` to accept and pass context_hints to task manager
- Context gathering fails silently to ensure agents work even without context
