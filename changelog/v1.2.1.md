# v1.2.1

Release date: 2026-01-17

## Fixes

### Doctor Command Improvements
- Fixed agent display to distinguish between Task tool agents and MCP background agents
- Task tool agents (Sisyphus, claude-reviewer, claude-scout) now show "(model managed by Claude Code)" since their model selection is controlled by Claude Code, not our configuration
- MCP background agents (oracle, analyst, librarian, etc.) continue to show their configured provider/model

### Model Name Format Fix
- Fixed model names from dash format (`claude-sonnet-4-5`) to dot format (`claude-sonnet-4.5`)
- Affected files: config schema, statusline formatter, agent definitions

### Version Detection Fix
- Fixed `oh-my-claude doctor --detail` showing "unknown" for version
- Now correctly reads version from installed package.json

### Beta Channel Detection
- Added `.beta-channel` marker file creation when installing from git dev branch
- Shows "(beta)" indicator in doctor command for local development installs
- Includes commit ref and branch info for tracking

## Technical Changes

- Added package.json copy to install directory for version detection
- Added git branch detection in installer for beta channel identification
- Improved agent categorization using provider type from config
- Added `prepare` script to package.json for GitHub tarball installs (requires Bun)
